# AI Reels — Cinema-Grade Storyboard Power Template
_Last updated: 2026-05-09_

## Summary
`storyboardBuilder.js` upgraded from a ~50-word grokPrompt to a full **13-section cinema-grade template** (~300+ words min). This dramatically improves GeminiGen output quality, consistency, and product accuracy.

## Template Sections (13 total)

| Section | Purpose |
|---|---|
| `[FORMAT]` | Aspect ratio + duration (was already present) |
| `[WORLD]` | **NEW** — Environment: location, time of day, atmosphere, lighting conditions |
| `[CHARACTER]` | Actor appearance, wardrobe, expression (was present, now has freeze rules) |
| `[PRODUCT]` | **NEW** — Container/label/content exact visual specs (shape, color, material, label text) |
| `[SCENE_FLOW]` | **NEW** — 3-beat timestamps: Enter / Mid / Peak with exact second markers |
| `[ACTION]` | Actor + product action — now has **Enter / Mid / Peak sub-beats** |
| `[CAMERA]` | Shot type, movement, angle |
| `[EFFECTS]` | **NEW** — Glow, particles, lens flares, overlays |
| `[COLOR_PALETTE]` | **NEW** — Named colors (e.g., "warm amber #D4A84B, deep teal #1A5F7A") |
| `[LIGHTING]` | Key/fill/rim light sources |
| `[VO]` | Voiceover script (2 sentences for 6s, 3 for 10s/15s) |
| `[MOOD]` | Emotional tone, pacing descriptor |
| `[RESTRICTIONS]` | Hard rules — **CHARACTER and PRODUCT both have freeze rules** |

## Key Upgrades vs Previous Template

### [ACTION] Sub-beats (Enter / Mid / Peak)
```
[ACTION]
Enter (0–2s): character walks into frame from left, product held at waist level
Mid (2–6s): character raises product to eye level, slow rotation reveals label
Peak (6–10s): freeze — product fills 60% of frame, character smiles at camera
```

### [PRODUCT] Section (new — prevents AI hallucination of product)
```
[PRODUCT]
Container: 150ml amber glass dropper bottle
Label: white matte label, black serif font, "SERUM X" centered
Content: golden oil visible through translucent glass
Placement: always right hand, label facing camera
```

### [WORLD] Section (new — grounds the scene)
```
[WORLD]
Location: minimalist studio with concrete wall
Time: golden hour, warm directional light from left
Atmosphere: clean, premium, slightly misty background
```

### [RESTRICTIONS] Freeze Rules
```
[RESTRICTIONS]
- CHARACTER: face visible and in focus for minimum 4s; no blurring
- PRODUCT: label must be readable at peak frame; no spinning past 90°
- No text overlays generated by AI
- Product color must match [PRODUCT] spec exactly — no reinterpretation
```

## Token / Length Changes
| Field | Before | After |
|---|---|---|
| `grokPrompt` word count | ~50 words | **300+ words minimum** |
| `maxTokens` for storyboard LLM call | ~2000 | **6000** |

## `technicalConfig` Schema Expansion
New fields added to the per-clip `technicalConfig` object returned by storyboardBuilder:

```js
{
  // existing
  aspectRatio, resolution, clipDuration, voScript, cameraWork,
  lightingSetup, colorGrading, characterDescription,

  // NEW
  characterDesign,   // detailed appearance/wardrobe for PRODUCT freeze rule
  productDesign,     // container/label/content exact specs
  worldBuilding,     // environment, time, atmosphere
  sceneFlow,         // { enter: string, mid: string, peak: string }
  effects,           // glow/particles/lens description
  colorPalette,      // named colors with hex codes
}
```

## sceneImageService Integration
`sceneImageService.js` now uses **all new technicalConfig fields** when building the gpt-image-2 prompt:
- `worldBuilding` → background/environment description
- `productDesign` → exact product visual in scene
- `effects` → post-process look
- `colorPalette` → dominant color constraint
- `sceneFlow.peak` → which moment to render (the "peak" beat)

## StoryboardClipCard UI Additions
`components/reels/StoryboardClipCard.tsx` now renders:
- **Scene Flow panel** — always visible; shows Enter / Mid / Peak beats with timestamps
- **World** — expandable section, shows environment description
- **Product** — expandable section, shows container/label/content specs
- **Effects** — expandable section, shows visual effects description
- **Color Palette** — expandable section, shows named swatches with hex codes
